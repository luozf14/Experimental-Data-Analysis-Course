二维直方图¶

  • Draw() 参数 Histograms' plotting options
  • 二维直方图 TH2 与一维直方图 TH1 类似，它能够用来展示一个两维度的分布。
  • 一维直方图将 x 轴划分成一系列间隔，而 y 轴则表示每个 bin 的计数。二维直方图将 x、y 轴都划分成一系列的间隔，则组成了 Nx * Ny 个 bin，z 轴用来表示每个 bin 的计数。然后每次填充的 (x,y) 数据则在相对应的 bin 上计数加 1
// (x, y)  坐标所在的 bin 计数加 1。
Int_t   Fill (Double_t x, Double_t y);

// 创建二维直方图
TH2I *h2 = new TH2I("h2", "",  100,0.0,10.0,  100,0.0,10.0 );

// 填充数据
// void SetBinContent(Int_t bin, Int_t, Double_t content)      x bin, y  bin, counter 
for (int i=1; i<=100; i++)
    for (int j=1; j<=100; j++)
      h2->SetBinContent(i, j, i*i+j*j);

//
 h2->GetXaxis()->SetTitle("xxx");
 h2->GetYaxis()->SetTitle("yyy");

h2->Draw("colz");
c1->Draw();
